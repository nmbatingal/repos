Getting Started

Mi. is a HTML/CSS template based on Bootstrap 3 framework. It allows to built impressive, better & faster, mobile friendly websites.

HTML Structure

This is the base of each HTML file:

<!-- Start of Header 
  ============================================= -->
<header id="header">
    ...
</header>
<!-- End of Header 
  ============================================= -->


<!-- Start of section
  ============================================= -->
  <section>
    ...
  </section>
<!-- End of section 
  ============================================= -->

<!-- div -->
  <div>
    ...
  </div>
<!-- div / End -->

<!-- Footer -->
<footer id="footer">
    ...
</footer>
<!-- Footer / End -->
